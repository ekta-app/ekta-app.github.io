get /organizationInstance/season/{season}
List organization instances for the given season (listOIsSeason)

Path parameters

season (required)
Path Parameter — Season to list organization instances for default: null

Query parameters

circuit (optional)
Query Parameter — Limit response to specific circuit(s) default: null

Return type

array[OrganizationInstance]

Example data

Content-Type: application/json
{
  "organizationId" : "organizationId",
  "elevated" : true,
  "verified" : true,
  "season" : "season",
  "_id" : "_id",
  "published" : true,
  "info" : {
    "website" : "website",
    "headerPicture" : "headerPicture",
    "primaryColor" : "primaryColor",
    "bio" : "bio",
    "blurb" : "blurb",
    "foundedYear" : "foundedYear",
    "affiliation" : "affiliation",
    "membersPicture" : "membersPicture",
    "contact" : {
      "snapchat" : "snapchat",
      "twitter" : "twitter",
      "facebook" : "facebook",
      "instagram" : "instagram",
      "email" : "email"
    },
    "name" : "name",
    "logo" : "logo",
    "location" : {
      "city" : "city",
      "state" : "state"
    },
    "secondaryColor" : "secondaryColor"
  }
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

List of organization instances
